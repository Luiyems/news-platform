Accesibilidad

Última actualización: 9 de abril de 2026

Nuestra compromiso

El rayo está comprometido con la accesibilidad web para todas las personas. Creemos que internet debe ser accesible para todos, independientemente de sus capacidades o dispositivos.

Este sitio ha sido diseñado siguiendo las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 nivel AA.

✓ WCAG 2.1 AA Compatible

Características de accesibilidad

🎯 Navegación por teclado

Todas las funciones son accesibles mediante el teclado. Usa Tab para moverte y Enter o Espacio para activar.

🔍 Contraste alto

Modo de alto contraste disponible para mejor visibilidad. Botón dedicado en la barra de herramientas.

📏 Texto redimensionable

Aumenta o reduce el tamaño de fuente según tus necesidades. A+ / A- buttons disponibles.

📱 Diseño responsivo

Se adapta a cualquier tamaño de pantalla: móvil, tablet y desktop.

🗣️ Tecnologias asistivas

Compatible con lectores de pantalla y dispositivos de entrada alternativos.

🌐 Idioma claro

Lenguaje simple y fácil de entender. Metadatos de idioma correctamente definidos.

Cómo usar las funciones de accesibilidad

Tamaño de fuente

Usa los botones A+ y A- en la barra de navegación para aumentar o disminuir el tamaño del texto. Tus preferencias se guardan automáticamente.

Modo alto contraste

Haz clic en el botón "Contraste" para activar el modo de alto contraste. Esto invierte los colores para mejor visibilidad.

Navegación por teclado

  • Tab: Moverse entre elementos
  • Shift + Tab: Moverse hacia atrás
  • Enter: Activar enlaces y botones
  • Espacio: Activar botones
  • Escape: Cerrar diálogos

Estándares seguidos

Este sitio cumple con los siguientes estándares de accesibilidad:

  • WCAG 2.1 Nivel AA
  • WAI-ARIA para aplicaciones ricas
  • HTML5 semántico
  • CSS responsivo

Excepciones conocidas

Algunas funciones externas pueden no cumplir completamente con los estándares:

  • Enlaces a noticias externas (fuentes originales)
  • Contenido embebido de terceros

Comentarios y sugerencias

Valoramos tu retroalimentación. Si encuentras dificultades de accesibilidad, por favor escríbenos con detalles:

  • URL de la página
  • Descripción del problema
  • Tecnología asistiva utilizada

Declaración de accesibilidad

El rayo se compromete a hacer este sitio accesible para todos los usuarios. Creemos que el derecho a la información es fundamental y debe ser accesible para personas con diversas capacidades.

Este compromiso incluye:

  • Mantenimiento continuo de accesibilidad
  • Capacitación del equipo en estándares web
  • Pruebas regulares con tecnologías asistivas
  • Priorización de problemas de accesibilidad en nuestro desarrollo
